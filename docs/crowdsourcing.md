# LPS — Crowd-sourced Package Suggestions

Users can suggest packages that are missing from LPS, upvote existing
suggestions, and submit via the web form or `curl`.  Registered users get a
higher rate limit and their name is attributed on their contributions.

---

## How it works

```text
User submits suggestion
        │
        ▼
  Validation layer
  (package name regex, known distro, length caps, honeypot)
        │
        ▼
  Duplicate check  ──dup──▶  409 "upvote it instead"
        │ no dup
        ▼
  Rate-limit check
  anon: 5/day (IP)  │  token: 20/day (user_id)
        │ OK
        ▼
  INSERT suggestions (status='pending')
        │
        ▼
  Other users upvote / downvote (toggle, one per IP)
        │
        ▼
  High-vote suggestions reviewed and merged by maintainers
```

## API endpoints

### Register and get a token

```bash
curl -s -X POST https://lps.eduluma.org/api/v1/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "email": "you@example.com",
    "display_name": "Your Name",
    "account_type": "individual"
  }'
```

**Response** (save your token — it is shown only once):

```json
{
  "id": 42,
  "display_name": "Your Name",
  "account_type": "individual",
  "token": "lps_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
  "created_at": "2026-04-22T00:00:00Z"
}
```

`account_type` is either `"individual"` or `"org"`.

---

### Submit a package suggestion

Authenticated (20 submissions/day):

```bash
curl -s -X POST https://lps.eduluma.org/api/v1/suggestions \
  -H "Authorization: Bearer lps_YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "package_name": "bat",
    "distro": "debian",
    "release": "bookworm",
    "install_cmd": "sudo apt install bat",
    "description": "A cat clone with syntax highlighting",
    "homepage_url": "https://github.com/sharkdp/bat"
  }'
```

Anonymous (5 submissions/day, no attribution):

```bash
curl -s -X POST https://lps.eduluma.org/api/v1/suggestions \
  -H "Content-Type: application/json" \
  -d '{ ... same body ... }'
```

**Successful response:**

```json
{ "id": 7, "status": "pending" }
```

**Error responses:**

| HTTP | Meaning                                                                                 |
| ---- | --------------------------------------------------------------------------------------- |
| 409  | A pending suggestion for this package/distro/release already exists — upvote it instead |
| 422  | Validation failed (invalid package name, unknown distro, etc.)                          |
| 429  | Daily rate limit exceeded                                                               |

---

### Browse pending suggestions

```bash
# First page (sorted by votes, highest first)
curl "https://lps.eduluma.org/api/v1/suggestions"

# Pagination
curl "https://lps.eduluma.org/api/v1/suggestions?limit=20&offset=20"
```

---

### Upvote / retract a vote

Calling the endpoint again on the same suggestion **retracts** the vote (toggle).

```bash
curl -s -X POST https://lps.eduluma.org/api/v1/suggestions/7/vote
```

**Response:**

```json
{ "vote_count": 4, "voted": true }
```

## Field reference

| Field          | Required | Rules                                                                 |
| -------------- | -------- | --------------------------------------------------------------------- |
| `package_name` | yes      | Lowercase, letters/digits/`.`/`-`/`_`/`+`, ≤ 100 chars                |
| `distro`       | yes      | One of: `debian` `ubuntu` `alpine` `arch` `fedora` `opensuse`         |
| `release`      | yes      | Letters/digits/`.`/`-`, ≤ 50 chars (e.g. `bookworm`, `noble`, `3.21`) |
| `install_cmd`  | yes      | ≤ 200 chars (standard template auto-generated on the web form)        |
| `description`  | yes      | One sentence, ≤ 200 chars                                             |
| `homepage_url` | no       | Must start with `https://` or `http://`                               |

---

## Install command templates

These are auto-generated by the web form. Override only for non-standard installs (e.g. a PPA).

| Distro          | Template                        |
| --------------- | ------------------------------- |
| debian / ubuntu | `sudo apt install <package>`    |
| alpine          | `apk add <package>`             |
| arch            | `sudo pacman -S <package>`      |
| fedora          | `sudo dnf install <package>`    |
| opensuse        | `sudo zypper install <package>` |

---

## Rate limits

| Caller       | Limit                 | Key        |
| ------------ | --------------------- | ---------- |
| Anonymous    | 5 suggestions / 24 h  | IP address |
| Token holder | 20 suggestions / 24 h | user_id    |

Votes are not rate-limited but are deduplicated: one vote per IP per suggestion.
